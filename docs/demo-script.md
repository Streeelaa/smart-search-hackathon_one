# Demo Script

## Цель демо

За 3-5 минут показать жюри, что система умеет:

1. исправлять опечатки;
2. работать с синонимами и разговорными формами;
3. применять hybrid retrieval;
4. менять выдачу на основе пользовательской истории;
5. давать измеримые метрики качества.

## Подготовка перед показом

1. Запустить backend: `uvicorn main:app --reload`
2. Запустить UI: `streamlit run streamlit_app.py`
3. Открыть вкладку `Demo` в Streamlit.
4. Если нужно, нажать `Reset demo state`.

## Сценарий 1. Опечатка

1. Открыть сценарий `Исправление Опечатки`.
2. Нажать `Загрузить сценарий`.
3. Перейти во вкладку `Search`.
4. Показать запрос `монитр`.
5. Показать `corrected_query = монитор`.
6. Показать релевантный результат с reasons.

Ключевая фраза:
Система не требует идеального ввода и исправляет типовые ошибки ещё до ранжирования.

## Сценарий 2. Синонимы

1. Открыть сценарий `Синонимы И Разговорные Формы`.
2. Загрузить сценарий.
3. Показать запрос `системник`.
4. Показать expanded_terms: `системный блок`, `пк`, `компьютер`.
5. Показать найденный системный блок.

Ключевая фраза:
Пользователь может искать неформальными словами, а система переводит их в язык каталога.

## Сценарий 3. Персонализация

1. Открыть сценарий `Персонализация До И После`.
2. Показать базовую выдачу по запросу `ноут`.
3. Нажать `Применить demo-события`.
4. Повторить поиск для `user-3`.
5. Показать, что reasons содержат признаки персонализации.
6. Перейти во вкладку `Profiles` и показать обновлённый профиль пользователя.

Ключевая фраза:
Система перестраивает выдачу по мере накопления действий конкретного заказчика.

## Сценарий 4. Метрики

1. Открыть вкладку `System`.
2. Показать `Hit@3`, `MRR@10`, `NDCG@10`.
3. Показать таблицу сравнения `keyword`, `semantic`, `hybrid`.
4. Объяснить, что на демо-датасете цифры близки, а на реальных данных сравнение станет содержательным.

Ключевая фраза:
Мы оцениваем не только наличие результата, но и качество верхушки выдачи и пригодность к персонализации.

## Завершение

Финальная фраза для защиты:
Мы построили не просто поиск по словам, а гибридную систему, которая объединяет correction, synonyms, semantic retrieval, rerank и динамическую персонализацию на основе поведения пользователя.
